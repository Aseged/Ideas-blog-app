**************************************** INSTALLATION MANUAL  ******************************

This product is tested using the following browsers:
	-Microsoft edge 25
	-Google Chrome 50
	-Mozilla Firefox 45

The Database and all necessary tables, along with stored procedures are created automatically. 

To start the application;

1.	Log into your VM
2.	Start the php server by typing  php –S 0:8080 –t web_coursework
(web_coursework must not be a sub-folder of another folder)
3.	Open available browser and type the server IP address followed by /log.php into the URL address bar.
4.	When prompted, please login or register.
(users must log in to enable the session to start)
